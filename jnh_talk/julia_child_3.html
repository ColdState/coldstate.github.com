If we've worked hard to create a shape, we might like to reuse it several times, without editing the magic numbers inside it. We can use translate to add an offset to the context.

Translate is a side-effectful thing. It's polite to leave the translation (which is called the transform matrix) where you found it. You could figure out the negative translate numbers to put it back, but it's easier to just use save and save.

(By the way, save and restore form a stack - save is like push restore is like pop.)
